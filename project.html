10-Week Programming Accelerator

I am currently working through a 10-week intensive programming accelerator alongside two of my cousins, guided by my uncle who designed the entire roadmap and curriculum for us. The goal we set for ourselves is straightforward, by the end of these 10 weeks, we want to reach the equivalent level of someone who has completed a one-year certificate in computer science.

The first week was not about writing code right away. It was about understanding how things actually work — how computers process information, how the internet functions behind the scenes, and how professional developers think and approach problems. From there we moved into the terminal, Git, GitHub, and the version control workflows that real software teams use every day. The week wrapped up with HTML, semantic web structure, and deploying a live website through GitHub Pages.

The project for this first week was to build a personal website using only HTML — the one you are reading right now. Every week comes with its own project, something concrete that puts what we learned that week into practice.

One week down. Nine to go.

What I am working toward over the next 9 weeks:

  • Learn JavaScript and CSS to build more interactive, functional, and visually polished websites
  • Understand how databases work and how to connect them to real projects
  • Be able to build and deploy a complete app from scratch, from the first line of code to a live working product
  • Learn how to design and build clean user interfaces that are both functional and visually strong
  • Develop the problem solving mindset and technical habits of a working software engineer
